Broken Access Control

Access Control is a security measure that enforces rules to determine what data or resources users are
allowed to access and what actions they can perform. When access control is not properly secured,
attackers can exploit vulnerabilities to access sensitive information and perform unauthorised actions.

In this lab, you will have access to the web application for Book Nook, where users can browse,
buy and wishlist books. However, the site has been deployed too early and the developers
were very sloppy. One has added the wrong button to the user's dashboard page, which takes authorised
users to the admin panel.

Task: Capture the Flag ! It is in a panel that only staff should be able to access.

Solution

> Log into the Book Nook account using the given credentials
> You can Inspect the page usually by Right-Clicking on the page
> Find the drop down for the "form" tag in the Developer Tools panel
> Double click the value attribute for the "input" tag and change its value from noadmin to admin
> Click the Manage Accounts button again and you should be taken to the admin panel
