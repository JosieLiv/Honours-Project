Security Misconfigurations

Security misconfigurations are extremely common and arise due to implementation flaws, incomplete settings
and unpatched software which can all be exploited quite easily by attackers. The misconfigurations can create
entry points for automated attacks which can lead to unauthorised users gaining full control of the system.

In this lab, you will have access to the web application for Book Nook, where users can browse, buy
and wishlist books. However, the site has been deployed too early and the developers were very sloppy.
Sensitive files are stored in a directory which has not been secured.

Task: Capture the Flag ! Find the directory containing sensitive files and the file containing the flag.
