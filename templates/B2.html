Cryptographic Failures

Cryptography is the practice of protecting data using algorithms that transform readable data into ciphertext
using cryptographic keys to ensure that only authorised users have access to it. However, these algorithms can
fail, exposing vulnerabilities to be exploited by attackers, seeking to access sensitive data.
In this lab, you will have access to the web application for Book Nook, where users can browse, buy
and wishlist books. However, the site has been deployed too early and the developers were very sloppy.
The developer testing out the login page left some key information in the source code.
