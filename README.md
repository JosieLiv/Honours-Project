Glasgow Caledonian University
20256/2026

DISCLAIMER:
This project was developed solely for educational purposes as part of an academic dissertation. It is not intended to perform malicious actions.

OVERVIEW:
The project creates an intentionally vulnerable web application designed to allow users to test their cybersecurity skills, through hands-on exploitation, in a safe environment. It covers the first five vulnerabilites from the OWASP Top 10 and offers three difficulty levels to choose from.

FEATURES:
- exploitation tasks based on the OWASP Top 10
- independent learning with optional hints and solutions
- dedicated mitigation page
- account creation with a leaderboard and progress tracker
- three difficulty levels: beginner, intermediate and advanced

TECHNOLOGY:
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: MySQL/MariaDB

CONTENTS:
- Home page
- Account page
- Lab difficulty page
- Mitigation page
- Beginner, intermediate, advanced task selection pages
- 15 task description pages
- multiple .html files for each task
- 2 styling pages
- 1 JavaScript file
- 2 python scripts for task solutions
- 2 text files for tasks
- 1 sql file with database queries
- python script for web app

The app.py will have to be run first, in a closed environment (such as a Kali Linux VM), to access the web application.
